# Tokyo Metro Dashboard

A modern, interactive dashboard for the Tokyo Metro system, featuring real-time-style status updates, station information, and a sleek design inspired by Tokyo's transit aesthetics.

## Features

- **Interactive Station Map**: View stations and lines with custom-designed logos.
- **Real-time Status**: Monitor the operational status of various metro lines.
- **Station Details**: Access information about specific stations, including line connections.
- **Modern UI**: Built with React, Tailwind CSS, and Motion for smooth animations.

## Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS 4
- **Animations**: Motion
- **Icons**: Lucide React
- **Backend**: Express (Node.js)
- **Language**: TypeScript

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/tokyo-metro-dashboard.git
   cd tokyo-metro-dashboard
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000`.
